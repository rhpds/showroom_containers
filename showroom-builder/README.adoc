== Overview

Build the container:

 docker build -t showroom-allinone .

Run the container:

 docker run -e GIT_REPO_URL=https://github.com/rhpds/aro-ilt -v $PWD/showroom/:/showroom/ -it showroom-allinone

Access the output on your host at ./showroom/www/index.html
